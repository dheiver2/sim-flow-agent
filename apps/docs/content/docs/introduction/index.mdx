Sim é um editor visual de workflows que permite construir aplicações com IA conectando blocos em uma tela. Arraste e solte componentes para criar chatbots, workflows de automação e pipelines de processamento de dados sem escrever código.

## O que Torna o Sim Poderoso

**Suporte a Múltiplos Modelos de IA** - Conecte-se à OpenAI, Anthropic, Google, Groq, Cerebras e modelos locais através do Ollama. Mude provedores sem reconstruir workflows.

**Mais de 60 Ferramentas Pré-construídas** - Gmail, Slack, Notion, Google Sheets, Airtable, Supabase, Pinecone e mais. Arquitetura extensível permite integração de ferramentas personalizadas.

**Execução Flexível** - Execute workflows via interface de chat, API REST, webhooks, jobs agendados ou dispare de sistemas externos.

**Deploy em Produção** - Implante como APIs, integre com sistemas existentes usando nosso SDK, ou incorpore como plugins. Monitoramento, logging e tratamento de erros integrados.

**Colaboração em Tempo Real** - Trabalhe simultaneamente com membros da equipe no mesmo workflow, como Google Docs para desenvolvimento de IA.

Conecte múltiplos modelos de IA, integre com mais de 60 serviços e implante aplicações prontas para produção através de uma interface visual intuitiva projetada especificamente para desenvolvimento de IA.

## Blocos de Construção Principais

**Blocos de Processamento**

- **Agente** - Execute inferência de modelo de IA com qualquer provedor LLM
- **API** - Conecte-se a endpoints REST e serviços externos  
- **Função** - Execute JavaScript personalizado para processamento de dados

**Blocos de Lógica**

- **Condição** - Crie lógica de ramificação baseada em avaliação de dados
- **Roteador** - Roteie caminhos de execução usando tomada de decisão com IA
- **Loop** - Itere sobre coleções sequencialmente
- **Paralelo** - Execute múltiplas operações simultaneamente

**Blocos de Saída**

- **Resposta** - Formate e retorne resultados finais do workflow
- **Avaliador** - Valide saídas contra critérios definidos
